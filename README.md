# Jeu de Dames (Checkers) - Java

Ce projet vise a developper une version du jeu de Dames (Checkers) en Java, avec une interface graphique JavaFX.

Le jeu de Dames est un jeu strategique a deux joueurs ou :
- les pions se deplacent en diagonale ;
- les captures se font en sautant par-dessus les pions adverses ;
- lorsqu'un pion atteint l'extremite opposee du plateau, il devient une dame, avec davantage de liberte de mouvement.

## Prerequis

- Java 17 (JDK 17)
- Maven (build et gestion des dependances)

## Construction du projet

```bash
mvn clean compile
```

## Execution (si une classe principale est configuree)

```bash
mvn exec:java
```
